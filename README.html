📋 Estructura del Examen

Parte teórica
50%
RI (7 pts) + BD (3 pts). Aprobar cada bloque por separado.
Parte práctica
50%
RI (3 pts) + BD (7 pts). Evaluación continua durante el cuatrimestre.
Mínimo para aprobar
×2
Hay que aprobar RI y BD de forma independiente en teórica Y práctica.
Patrón histórico
≈
La parte RI es casi idéntica cada año. BD varía más entre convocatorias.

🗂️ Contenidos por Bloque

🔍 Recuperación de Información (7 pts)

  • Índice invertido — AND, OR, NOT, phrase queries, optimización
  • TF-IDF, modelo vectorial, similitud coseno, normalización
  • Métricas — P@k, MAP, MRR, NDCG, evaluación por clicks
  • PageRank (matriz P, teleportación) + HITS
  • BIM y BM25 — modelos probabilísticos
  • Clasificación — Naive Bayes, Rocchio, kNN
  • Preprocesado, MapReduce distribuido, crawling

🗄️ Bases de Datos NoSQL (3 pts)

  • 4 modelos NoSQL — clave-valor, documentos, columnas, grafos
  • Teorema CAP — centralizado vs distribuido
  • ACID vs BASE, quórums, consistencia eventual
  • Replicación maestro-esclavo vs peer-to-peer, Vector Stamp
  • Sharding — hashed vs ranged, shard key
  • MongoDB cluster — mongod, mongos, rs.initiate, sh.addShard, pipeline
  • Neo4J / Cypher — MATCH, MERGE, WITH, collect, UNION, OPTIONAL MATCH
  • BD objeto-relacionales — SQL:1999, CREATE TYPE, ONLY()

🚀 Cómo usar esta wiki

1
Clona el repositorio y ábrelo en Obsidian

Importa la carpeta XINE/ como vault en Obsidian. Los enlaces [[]] funcionan automáticamente.

2
Lee primero los documentos de entrada

Empieza por wiki/temas/00-criterios-evaluacion.md y wiki/temas/00-examenes-reales.md. Todo lo demás sale de ahí.

3
Navega por el grafo de conceptos

Usa wiki/index.md como mapa. Está organizado por bloques con prioridad de examen (⭐⭐⭐ = sale seguro).

4
Usa Claude como tutor con el CLAUDE.md configurado

Abre Claude Code en la raíz del repo. El CLAUDE.md tiene un algoritmo BFS de routing para que Claude navegue la wiki de forma eficiente. Puedes preguntarle sobre cualquier tema, pedir que te examine o que ingeste nuevos materiales.
